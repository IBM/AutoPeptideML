Automating Peptide Bioactivity Prediction: User Guide

Introduction to User Guide

The aim of this user guide is to provide assistance in the basic use of the web server. Its contents are divided into 5 sections:

  • Algorithm: provides a description of the computational method and a describes its default configuration for the web-server application.
  • Build Predictor: describes the corresponding utility of the web-server, indicates how it should be used, what the format of the inputs ought to be, and provides some examples to allow for simple experimentation.
  • Predict: describes the corresponding utility of the web-server, indicating how the inputs should be formatted, what to expect of the outputs, and provides some examples to allow for simple experimentation.
  • Command-line application: Guide for installation and of the command-line application.
  • FAQs: a collection of answers to the commonly asked questions.
  • GitHub Repository: link to the project GitHub repository where the source code is hosted. Please report any problems with the software as an issue there.
  • How to cite: instructions in how to cite the tool.

    • The tool has been developed by Raul Fernandez-Diaz as a PhD Student within ShieldsLab at University College Dublin and IBM Research Dublin. The web-server is being hosted by IBM Cloud.
